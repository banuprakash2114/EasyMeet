# EasyMeet - Video Call App

EasyMeet is a simple and user-friendly video call application that allows users to create and join meetings using dynamic meeting URLs.

## Features:
- User registration and login with real-time session management.
- Room-based video call functionality with dynamic meeting URLs.
- Local storage integration for persisting user details.

## How to Use:
1. Register an account or log in to your existing account.
2. Create a new meeting or join an existing one by entering the meeting ID.

## Technologies Used:
- Backend: Spring Boot, REST API, CommandLineRunner for data initialization.
- Frontend: JavaScript, HTML, CSS.
- Tools: IntelliJ IDEA, Postman for API testing.
